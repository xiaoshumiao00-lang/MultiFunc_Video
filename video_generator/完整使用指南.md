# 抖音短视频批量生成工具 - 完整使用指南

## 一、项目概述

这是一套本地部署的抖音短视频批量生成工具，完全免费，支持：

| 功能模块 | 技术方案 | 特点 |
|---------|---------|------|
| 🎨 图片生成 | FLUX.1 Dev (ComfyUI秋叶整合包) | 中文字渲染强，风格多样 |
| 🔊 语音合成 | Edge-TTS / GPT-SoVITS | 免费快速 / 语音克隆 |
| 📝 字幕生成 | 自动时间轴对齐 | ASS格式，完美同步 |
| 🎬 视频合成 | FFmpeg | 高质量输出，支持水印 |

## 三、准备清单

### ✅ 1. ComfyUI 秋叶整合包（必须）

**用途**：生成高质量AI图片

**下载**：
- B站搜索"秋叶comfyui整合包"
- 或访问 https://github.com/comfyanonymous/ComfyUI

**安装**：
1. 解压到本地目录（如 `D:\FLUX Redux`）
2. 双击启动 `run.bat`
3. 浏览器访问 http://127.0.0.1:8188 确认运行

**模型配置**：
```

├── unet/
│   └── flux1_dev_fp8.safetensors    # UNet模型
├── vae/
│   └── ae.safetensors               # VAE模型
└── clip/
    ├── t5xxl_fp8_e4m3fn.safetensors # T5 CLIP
    └── clip_l.safetensors           # CLIP L
```

## 四、安装步骤

### 1. 克隆/下载项目

```bash
cd c:/Users/Administrator/WorkBuddy/20260324112324
cd video-generator
```

### 2. 安装Python依赖

```bash
pip install -r requirements.txt
```

**主要依赖**：
- edge-tts（免费语音合成）
- httpx（API调用）
- moviepy（视频处理，可选）
- pillow（图片处理）

### 3. 配置 config.py

编辑 `config.py` 中的关键配置：

```python
# ============ ComfyUI FLUX 设置 ============
COMFYUI_SETTINGS = {
    "api_url": "http://127.0.0.1:8188",  # ComfyUI地址
    "model": "flux1_dev_fp8.safetensors",
    "vae": "ae.safetensors",
    "clip1": "t5xxl_fp8_e4m3fn.safetensors",
    "clip2": "clip_l.safetensors",
    "steps": 25,
}

# ============ 语音设置 ============
TTS_MODE = "edge"  # 或 "gpt_sovits"

# GPT-SoVITS配置（如果使用）
GPT_SOVITS_SETTINGS = {
    "api_url": "http://127.0.0.1:9880",
    "ref_audio_path": "voices/my_voice.wav",  # 你的参考音频
    "prompt_text": "参考音频对应的文字内容",
}

# ============ 视频设置 ============
VIDEO_SETTINGS = {
    "width": 1080,      # 宽度
    "height": 1920,     # 高度（9:16竖屏）
    "fps": 30,
    "bitrate": "10M",
}
```

## 五、使用方法

### 🚀 方式一：命令行单条生成

#### Edge-TTS 模式（快速免费）

```bash

# 指定主题和水印
python main.py --text "文案内容" --theme "考研攻略篇"

# 指定音色
python main.py --text "文案" --voice zh-CN-YunxiNeural
```

```bash
# 1. 先启动GPT-SoVITS服务

python main.py --text "你的文案" --tts gpt_sovits
```

## 八、常见问题

### Q1: ComfyUI连接失败
```
[ERROR] ComfyUI is not running
```
**解决**：确保秋叶整合包已启动，访问 http://127.0.0.1:8188 确认

### Q2: 图片生成慢
**解决**：
- 降低步数（steps从25降到15-20）
- 使用FP8量化模型
- 确保显卡驱动是最新

### Q3: 语音合成失败
**Edge-TTS**: 检查网络连接
**GPT-SoVITS**: 确保服务已启动在9880端口

### Q4: 视频合成失败
**解决**：
- 检查FFmpeg是否安装：`ffmpeg -version`
- 检查图片/音频路径是否正确
- 查看错误日志
